# InfoShots

InfoShots is a lightweight Flutter application that delivers random summarized Wikipedia articles in a clean, distraction-free format.
It is designed for quick learning by presenting concise insights while allowing users to explore full articles when needed.


## Features

* Random Knowledge Feed
* Fetches random Wikipedia articles using the Wikipedia API.
*	Summarized Content
*	Displays short, readable summaries for quick consumption.
*	Bookmark Articles
*	Save your favorite articles locally for later reading.
*	Read Full Article
*	Open the complete Wikipedia page in an in-app WebView via the “Read More” button.
*	Fast & Reactive UI
*	Built with Riverpod for efficient and scalable state management.
*	Offline Bookmark Storage
*	Uses SQLite (sqflite) to persist bookmarked articles locally.


## Tech Stack

**Frontend**
*	Flutter
**State Management**
*	Riverpod
**API**
* Wikipedia API(wikimedia)
**Local Database**
*	sqflite
**Other Packages**
*	HTTP (API calls)
*	WebView (for full article view)


## Installation

1. Clone the repository

git clone https://github.com/shamimarqam/infoshots.git
cd infoshots

2. Install dependencies

flutter pub get

3. Run the app

flutter run
